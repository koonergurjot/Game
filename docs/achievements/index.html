Achievement tracker

Understand how achievements unlock, sync, and display across web, launcher, and mobile surfaces.

How it works

Every catalog game integrates with the Gurjot Achievement Service (GAS). When you fulfill the conditions for an achievement, the game posts a signed payload to GAS which syncs across devices within seconds.

Offline sessions queue unlocks locally and sync once you reconnect.

Tracking progress

  • On the web

    Open the profile overlay (top right) and switch to the Achievements tab for per-game breakdowns.

  • In the launcher

    Use the Achievements widget to filter by rarity, completion percentage, and friends who have unlocked it.

Developer integration tips

  1. 1. Define achievements

    Create a achievements.json file in your game directory with ID, name, description, and point values.

  2. 2. Post unlocks

    Use window.GG.achievements.unlock(id) in your game code. The helper handles retries and offline queues.

  3. 3. Test locally

    Enable the Achievements toggle in the dev overlay (Ctrl+Shift+A) to inspect unlock logs while you play.

FAQ

Do achievements reset? Never. Seasonal events add limited-time goals, but core achievements remain.

Can I delete progress? Yes—visit the privacy page in the launcher to wipe stored data per game.

Need to report a bug? Email achievements@gurjotsgames.com.
