Install our desktop companion app for silky-smooth capture, controller remapping, and offline play.

Why use the launcher?

  • Offline ready

    Pre-cache your installed games and play without a network connection.

  • Controller mapper

    Map Xbox, PlayStation, and Switch controllers with per-game profiles.

  • Performance tools

    Unlock frame pacing analytics, GPU overlays, and direct log export.

  • VR experiment hub

    Stream prototype OpenXR builds directly to supported headsets.

Setup steps

  1. 1. Download & install

    Grab the latest installer for your platform from the releases page above. Windows and macOS packages are notarized; Linux builds ship as AppImage.

  2. 2. Sign in

    Use the same Gurjot ID as the web portal. Your achievements, save data, and quests carry over automatically.

  3. 3. Sync libraries

    Choose which games to keep offline, then toggle auto-update to pull fresh builds overnight.

Beta notes

The launcher is currently in public beta. Expect frequent updates while we dial in VR streaming and controller haptics.

Encountered a bug? Capture logs via the diagnostics drawer and file an issue with your OS version, GPU, and reproduction steps.

Need technical support? Email launcher@gurjotsgames.com with your diagnostics bundle.
