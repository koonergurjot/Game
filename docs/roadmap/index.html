Platform roadmap

    Live quest widget embedded in the play shell so you never lose sight of seasonal objectives.

    Desktop app with offline caching, controller mapping, and VR experiment streaming.

In development

    Encrypted storage upgrades plus selective sync per device class.

  3. 3. Advanced mod tools

    Steam Workshop-inspired pipelines for community levels with moderation hooks.
